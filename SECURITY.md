# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via:

- **GitHub Security Advisories**: Use the "Report a vulnerability" button on the Security tab
- **Email**: Private disclosure is preferred for critical issues

Please do not disclose security issues publicly until a fix is available.
